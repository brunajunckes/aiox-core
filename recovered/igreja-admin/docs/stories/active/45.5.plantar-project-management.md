## 📊 API Endpoints

```
GET    /api/plantar                             — List projects
POST   /api/plantar                             — Create project
GET    /api/plantar/[id]                        — Get project detail
PUT    /api/plantar/[id]                        — Update project
DELETE /api/plantar/[id]                        — Delete project

GET    /api/plantar/[id]/milestones             — List milestones
POST   /api/plantar/[id]/milestones             — Create milestone
PUT    /api/plantar/[id]/milestones/[mId]       — Update milestone
DELETE /api/plantar/[id]/milestones/[mId]       — Delete milestone

GET    /api/plantar/[id]/updates                — List project updates
POST   /api/plantar/[id]/updates                — Create update (with photos)
GET    /api/plantar/[id]/updates/[uId]          — Get update detail

GET    /api/plantar/[id]/team                   — List team members
POST   /api/plantar/[id]/team                   — Add team member
DELETE /api/plantar/[id]/team/[userId]          — Remove member

GET    /api/plantar/[id]/funding                — Funding progress + donors
POST   /api/plantar/[id]/fund                   — Donate (Church.sol integration)
```
