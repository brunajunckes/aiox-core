# Story 45.6: Semear — Community Idea Voting & Validation

**Story ID:** 45.6  
**Depends On:** Stories 45.1-45.3

## 🎯 Acceptance Criteria

- [x] Submit idea (title, description, category, resources_needed) — authenticated
- [x] List ideas (paginated, filter by status/category)
- [x] Get idea detail (votes, discussions, author info)
- [x] Edit idea — author+
- [x] Delete idea — author/moderator+
- [x] Idea status: submitted/approved/rejected/implemented

- [x] Voting system: Upvote/downvote per user (unique constraint)
- [x] Vote count display (net votes = upvotes - downvotes)
- [x] Vote tracking (who voted when)
- [x] Voting permissions: authenticated users only

- [x] Discussion threads per idea
- [x] Comments on discussions
- [x] Discussion reply notifications

- [x] Contributor recognition (list contributors to an idea)
- [x] Contribution badges (based on vote count)

- [x] Idea → Project conversion (moderator+)
- [x] Conversion workflow documented
- [x] Status change notification

- [x] Search ideas by title/description
- [x] Filter by status, category, votes
- [x] Sort by newest/trending/most_voted

- [x] React components: IdeaGallery, IdeaDetail, SubmitForm, DiscussionThread, VotingUI, ContributorBadges
- [x] >70% test coverage
- [x] Lighthouse ≥80

## 📊 API Endpoints

```
GET    /api/semear                              — List ideas
POST   /api/semear                              — Submit idea
GET    /api/semear/[id]                         — Get idea detail
PUT    /api/semear/[id]                         — Edit idea
DELETE /api/semear/[id]                         — Delete idea
PUT    /api/semear/[id]/status                  — Change status (admin)

POST   /api/semear/[id]/vote                    — Upvote/downvote
GET    /api/semear/[id]/votes                   — Get vote count

GET    /api/semear/[id]/discussions             — List discussions
POST   /api/semear/[id]/discussions             — Add discussion
GET    /api/semear/[id]/discussions/[dId]       — Get discussion thread
POST   /api/semear/[id]/discussions/[dId]/reply — Reply to discussion

GET    /api/semear/[id]/contributors            — Get contributor list
PUT    /api/semear/[id]/convert-to-project      — Convert to project (admin)

GET    /api/semear/trending                     — Trending ideas
```

---

## 📝 Notes

- One vote per user per idea (UNIQUE constraint: idea_id, user_id)
- Voting opens discussion and engagement
- Approved ideas can become Plantar projects
- Contributor recognition motivates participation
- Real-time vote count updates (Story 46.1)
