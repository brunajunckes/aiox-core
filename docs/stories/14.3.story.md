# Story 14.3: Contributor Stats

## Objective

Provide a CLI command to show contributor stats from git log with filtering and top-N support.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** to see contributor statistics,
**so that** I can understand team contribution patterns.

## Acceptance Criteria

- [x] `aiox contributors` shows contributor list with commit counts
- [x] `aiox contributors --format json` output as JSON
- [x] `aiox contributors --since 30d` filter by time period
- [x] `aiox contributors --top N` show top N contributors
- [x] Uses `git shortlog -sn` internally
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/contributors.test.js`
