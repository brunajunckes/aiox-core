# Story 10.4: CLI Session Replay & History

## Objective

Provide an `aiox history` command that tracks CLI command usage in a JSONL file and supports viewing, searching, exporting, and clearing history.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using AIOX CLI frequently,
**I want** to view and search my command history,
**so that** I can recall past commands and export usage data.

## Acceptance Criteria

- [x] `aiox history` shows last 20 commands from `.aiox/command-history.jsonl`
- [x] `aiox history --limit N` shows last N commands
- [x] `aiox history --search <term>` filters history by keyword (command + args)
- [x] `aiox history --clear` clears history
- [x] `aiox history --export` exports history as JSON to stdout
- [x] Each entry format: `{ command, args, timestamp }`
- [x] Search is case-insensitive
- [x] Handles missing/empty history file gracefully
- [x] Table output with numbered rows
- [x] Command registered in bin/aiox.js

## Dev Notes

- History file: `.aiox/command-history.jsonl` (one JSON object per line)
- Search matches against both command name and args joined as string
- Export outputs full JSON array (not JSONL) for easy consumption
- Creates `.aiox/` directory automatically when appending
