# Story 20.2: Migration Guide Generator

## Objective

Generate migration guides by detecting breaking changes in git history.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** automatic migration guide generation,
**so that** I can document breaking changes between versions.

## Acceptance Criteria

- [x] `aiox migrate-guide` generates migration guide from breaking changes in git log
- [x] `aiox migrate-guide --from v5.0.0 --to v6.0.0` specific version range
- [x] `aiox migrate-guide --format markdown` as markdown
- [x] `aiox migrate-guide --format json` as JSON
- [x] `aiox migrate-guide --output docs/migration.md` write to file
- [x] Detects: BREAKING CHANGE in commit messages, removed exports, changed APIs
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
