# Story 14.4: Changelog Auto-Generator

## Objective

Provide a CLI command to auto-generate changelogs from conventional commits, grouped by category.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** to auto-generate changelogs from conventional commits,
**so that** I can produce release notes without manual effort.

## Acceptance Criteria

- [x] `aiox auto-changelog` generates changelog from conventional commits since last tag
- [x] `aiox auto-changelog --since v5.0.0` from specific tag
- [x] `aiox auto-changelog --write` appends to CHANGELOG.md
- [x] `aiox auto-changelog --format json` output as JSON
- [x] Groups: Features, Bug Fixes, Documentation, Testing, Chores
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/auto-changelog.test.js`
