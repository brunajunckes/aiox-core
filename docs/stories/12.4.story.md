# Story 12.4: Governance Report Generator

## Objective

Provide a CLI command to generate a governance report aggregating code ownership, test coverage, dependency count, security scan, and license compliance.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** a governance report that aggregates all quality metrics,
**so that** I can get a single view of project health and compliance.

## Acceptance Criteria

- [x] `aiox governance` generates a governance report
- [x] `aiox governance --format json` outputs as JSON
- [x] `aiox governance --export` saves to `.aiox/governance-report.json`
- [x] `aiox governance --ci` exits 1 if any governance check fails
- [x] Aggregates: git log (contributors), package.json (deps), coverage report, secret scan, license compliance
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/governance.test.js`
