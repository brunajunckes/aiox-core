Restore Igreja-nas-casas local configuration and make it accessible via Docker with health checks on ports 80/443.

**As a** Igreja platform maintainer,

**so that** the platform is accessible locally on HTTP/HTTPS with working health checks.

- [x] Extract Igreja configs from `/root/recovered/igreja-configs` (from GDrive backup)
- [x] Build Docker image for Igreja service
