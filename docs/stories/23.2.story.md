# Story 23.2: Package Size Analyzer

## Objective

Analyze package size with install/publish estimates, directory breakdowns, and ASCII treemap.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a package size analyzer,
**so that** I can monitor and control the size of my project.

## Acceptance Criteria

- [x] `aiox pkg-size` shows package size: install size, publish size, file count
- [x] `aiox pkg-size --details` shows per-directory breakdown
- [x] `aiox pkg-size --format json` outputs as JSON
- [x] `aiox pkg-size --limit 1mb` warns if over limit
- [x] `aiox pkg-size --treemap` shows ASCII treemap visualization
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
