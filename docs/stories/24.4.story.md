# Story 24.4: Uptime Tracker

## Objective

Show project uptime stats based on git history with calendar heatmap and streak tracking.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** an uptime tracker,
**so that** I can visualize project activity and commit patterns over time.

## Acceptance Criteria

- [x] `aiox uptime` shows project uptime stats (first commit date, days active, commit frequency)
- [x] `aiox uptime --format json` outputs as JSON
- [x] `aiox uptime --calendar` shows ASCII calendar heatmap of commit activity (last 3 months)
- [x] `aiox uptime --streak` shows current and longest commit streak
- [x] Uses git log for data
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
