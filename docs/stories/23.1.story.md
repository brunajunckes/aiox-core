# Story 23.1: Git Diff Analyzer

## Objective

Analyze uncommitted git changes with impact scoring, risk assessment, and reviewer suggestions.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a git diff analyzer,
**so that** I can understand the scope and risk of my changes before committing.

## Acceptance Criteria

- [x] `aiox diff-analyze` analyzes uncommitted changes: files changed, lines added/removed, impact score
- [x] `aiox diff-analyze --staged` analyzes staged changes only
- [x] `aiox diff-analyze --format json` outputs as JSON
- [x] `aiox diff-analyze --risk` provides risk assessment (high if >500 lines or >10 files)
- [x] `aiox diff-analyze --suggest-reviewers` suggests reviewers based on git blame
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
