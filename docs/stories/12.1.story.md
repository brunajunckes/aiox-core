# Story 12.1: Secret Scanner

## Objective

Provide a CLI command to scan project files for leaked secrets (API keys, tokens, passwords) and optionally replace them with placeholders.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** a secret scanner that detects leaked credentials in my codebase,
**so that** I can prevent accidental exposure of sensitive data.

## Acceptance Criteria

- [x] `aiox secrets scan` scans project files for leaked secrets
- [x] `aiox secrets scan --fix` replaces found secrets with placeholders
- [x] `aiox secrets scan --format json` outputs findings as JSON
- [x] Detects: API_KEY=, SECRET=, PASSWORD=, TOKEN=, private_key, Bearer, AWS keys (AKIA...), base64 secrets
- [x] Ignores: node_modules/, .git/, .env.example, *.test.js
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/secrets.test.js`
