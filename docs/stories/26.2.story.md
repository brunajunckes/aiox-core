# Story 26.2: ASCII Chart Generator

## Objective

Provide an ASCII chart generator via `aiox chart` that renders bar, line, and pie charts directly in the terminal from user-provided data strings.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using AIOX,
**I want** to generate ASCII charts from the CLI,
**so that** I can visualize data trends and distributions without leaving the terminal.

## Acceptance Criteria

- [x] `aiox chart bar --data "Mon:5,Tue:8,Wed:3,Thu:12,Fri:7"` renders ASCII bar chart
- [x] `aiox chart line --data "1,3,2,5,4,8,6"` renders ASCII line chart
- [x] `aiox chart pie --data "JS:60,TS:25,MD:15"` renders ASCII pie chart (percentage bars)
- [x] `aiox chart --title "Build Times"` adds title
- [x] `aiox chart --width 60` sets chart width
- [x] `aiox chart --format json` outputs chart data as JSON
- [x] Shows help when no subcommand or invalid subcommand
- [x] Handles edge cases: empty data, zero values, single point
- [x] Zero external npm dependencies
- [x] Command registered in bin/aiox.js
- [x] 15+ tests in tests/cli/chart.test.js

## Scope

**IN:**
- Bar, line, pie chart rendering
- Data parsing (key:value and numeric)
- Title, width, format options

**OUT:**
- Interactive chart editing
- Color/ANSI in charts
- File-based data input
