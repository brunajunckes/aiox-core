# Story 28.2: Database Model Generator

## Objective

Generate database model files with validation and defaults from a CLI command, streamlining data layer development.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** to generate model files with field validation from a single command,
**so that** I can quickly scaffold data models with consistent patterns.

## Acceptance Criteria

- [x] `aiox gen-model <name> --fields "id:number,name:string,email:string"` generates model
- [x] `aiox gen-model --list` lists generated models
- [x] `aiox gen-model <name> --timestamps` adds createdAt/updatedAt fields
- [x] Generates model.js with validation + model.test.js
- [x] Shows help with no args or --help
- [x] Validates model name is provided
- [x] Supports types: string, number, boolean, date, object, array
- [x] Zero external npm dependencies
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing

## Scope

**IN:**
- Model generation with typed fields
- Validation function generation
- Timestamp support (createdAt/updatedAt)
- Listing generated models

**OUT:**
- Database migration generation
- ORM integration
- Relationship definitions
