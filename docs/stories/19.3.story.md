# Story 19.3: Command Reference Generator

## Objective

Generate a complete command reference for all AIOX CLI commands in multiple formats.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a command reference generator,
**so that** I can produce up-to-date documentation of all CLI commands.

## Acceptance Criteria

- [x] `aiox ref` generates complete command reference (text)
- [x] `aiox ref --format markdown` outputs as markdown
- [x] `aiox ref --format json` outputs as JSON
- [x] `aiox ref --output <file>` writes to file
- [x] `aiox ref --category <cat>` filters by category
- [x] Categorizes all commands into logical groups
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing

## Scope

**IN:** ref command, text/markdown/json formats, file output, category filter
**OUT:** HTML format, auto-publish
