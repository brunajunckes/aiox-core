# Story 21.2: Dependency Graph Visualizer

## Objective

Show dependency tree from package.json with ASCII output and circular dependency detection.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a dependency graph visualizer,
**so that** I can understand project dependencies and detect circular require chains.

## Acceptance Criteria

- [x] `aiox dep-graph` shows dependency tree from package.json
- [x] `aiox dep-graph --depth N` limits tree depth
- [x] `aiox dep-graph --dev` includes devDependencies
- [x] `aiox dep-graph --format json` outputs as JSON
- [x] `aiox dep-graph --circular` detects circular dependencies in local require() calls
- [x] ASCII tree output for terminal
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
