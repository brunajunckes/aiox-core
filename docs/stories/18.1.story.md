# Story 18.1: Test Generator

## Objective

Auto-generate Jest test skeletons for JS modules by parsing module.exports.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a CLI command that generates test skeletons from module exports,
**so that** I can quickly scaffold tests for untested modules.

## Acceptance Criteria

- [x] `aiox test-gen <file>` auto-generates Jest test skeleton for a JS module
- [x] `aiox test-gen <file> --output <path>` writes to specific file
- [x] Parses module.exports to discover exported functions
- [x] Generates describe/it blocks with TODO placeholders for each function
- [x] `aiox test-gen --list <dir>` lists untested modules (no matching .test.js)
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing

## Scope

**IN:** test-gen command, export parsing, skeleton generation, --list mode
**OUT:** TypeScript support, complex class parsing
