# Story 19.1: Man Page Generator

## Objective

Generate formatted man-page-style help for all AIOX CLI commands.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a man page generator for AIOX commands,
**so that** I can quickly look up detailed usage for any command.

## Acceptance Criteria

- [x] `aiox man` shows formatted help for all commands (overview)
- [x] `aiox man <command>` shows detailed help for specific command
- [x] `aiox man --generate` generates man page files at `.aiox/man/`
- [x] `aiox man --list` lists all available man pages
- [x] Reads command metadata from internal registry
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing

## Scope

**IN:** man command, overview, per-command pages, file generation, list
**OUT:** nroff format, system man path integration
