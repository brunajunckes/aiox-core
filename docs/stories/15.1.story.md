# Story 15.1: Task Runner with Dependencies

## Objective

Implement a task runner with DAG-based dependency resolution, allowing users to define, manage, and execute tasks with dependency chains via `aiox tasks`.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** a task runner that resolves dependencies as a DAG,
**so that** I can define build/test/lint pipelines that execute in correct order.

## Acceptance Criteria

- [x] `aiox tasks list` lists all defined tasks from `.aiox/tasks.yaml`
- [x] `aiox tasks run <name>` runs a task with dependency resolution
- [x] `aiox tasks add <name> "<command>"` adds a task
- [x] `aiox tasks add <name> "<command>" --deps "task1,task2"` adds with dependencies
- [x] `aiox tasks remove <name>` removes a task
- [x] Tasks are DAG — circular dependencies are detected and rejected
- [x] Task format: `{ name, command, deps: [], description }`
- [x] 15+ tests in `tests/cli/tasks.test.js`
- [x] Command registered in `bin/aiox.js`

## Tasks / Subtasks

- [x] Task 1: Implement YAML-like parser and serializer for tasks file
- [x] Task 2: Implement DAG cycle detection and topological sort
- [x] Task 3: Implement task execution with dependency resolution
- [x] Task 4: Implement list/add/remove subcommands
- [x] Task 5: Register command in bin/aiox.js
- [x] Task 6: Write 15+ tests covering all functions
