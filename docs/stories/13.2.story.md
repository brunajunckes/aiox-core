# Story 13.2: Project Init Templates

## Objective

Provide a CLI command to scaffold new projects with AIOX boilerplate using configurable templates.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** to initialize new projects from templates,
**so that** I can quickly bootstrap projects with the right structure.

## Acceptance Criteria

- [x] `aiox init-project <name>` creates new project directory with AIOX boilerplate
- [x] `aiox init-project <name> --template api` uses API template
- [x] `aiox init-project <name> --template cli` uses CLI template
- [x] `aiox init-project <name> --template fullstack` uses fullstack template
- [x] `aiox init-project --list-templates` shows available templates
- [x] Creates: package.json, .gitignore, README.md, src/ structure, tests/, .aiox/ config
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/init-project.test.js`
