# Story 13.3: Interactive Config Wizard

## Objective

Provide a CLI command that configures AIOX project settings interactively or with defaults.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** a setup wizard that configures my project,
**so that** I can quickly configure AIOX without editing files manually.

## Acceptance Criteria

- [x] `aiox setup` starts interactive wizard (when TTY available)
- [x] `aiox setup --defaults` applies default config non-interactively
- [x] `aiox setup --show` shows current setup summary
- [x] `aiox setup --reset` resets to defaults
- [x] Configures: project name, default agent, git integration, test command, lint command
- [x] Stores in `.aiox/setup.json`
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/setup.test.js`
