# Story 23.3: Environment Variable Manager

## Objective

Manage .env files: list vars, validate against .env.example, diff, and generate.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** an environment variable manager,
**so that** I can ensure my .env files are complete and consistent.

## Acceptance Criteria

- [x] `aiox env-vars list` lists all env vars from .env files
- [x] `aiox env-vars check` validates .env against .env.example (finds missing vars)
- [x] `aiox env-vars diff` diffs .env vs .env.example
- [x] `aiox env-vars generate` generates .env from .env.example with defaults
- [x] `aiox env-vars --format json` outputs as JSON
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
