# Story 11.4: Environment Info & Diagnostics Export

## Objective

Provide a CLI command to display environment information, run health checks, and export diagnostics reports.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** an environment info command that shows system details and runs health checks,
**so that** I can diagnose issues and share environment details for support.

## Acceptance Criteria

- [x] `aiox env` shows full environment info (Node, npm, git, OS, shell, AIOX version, plugins, stories)
- [x] `aiox env --export` exports as JSON to stdout
- [x] `aiox env --check` runs basic health checks (node >=18, git, npm, .aiox-core)
- [x] `aiox env --report` generates support report file at `.aiox/env-report.json`
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/env.test.js`
