# Story 13.4: CLI Output Formatter & Themes

## Objective

Provide a CLI command to manage output themes that control formatting, color, and verbosity across all AIOX commands.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** to choose output themes for CLI commands,
**so that** I can customize how information is displayed.

## Acceptance Criteria

- [x] `aiox theme` shows current theme
- [x] `aiox theme list` shows available themes (default, minimal, verbose, json)
- [x] `aiox theme set <name>` sets active theme
- [x] `aiox theme set json` makes all commands output JSON by default
- [x] Themes stored in `.aiox/theme.json` as `{ active: "default" }`
- [x] Themes affect: table formatting, color usage, verbosity level
- [x] Utility `getTheme()` function other commands can import
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/theme.test.js`
