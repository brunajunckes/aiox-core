# Story 21.1: Code Complexity Analyzer

## Objective

Analyze JS files and report cyclomatic complexity per function with filtering and formatting options.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a code complexity analyzer,
**so that** I can identify overly complex functions and prioritize refactoring.

## Acceptance Criteria

- [x] `aiox complexity` analyzes all JS files, reports cyclomatic complexity per function
- [x] `aiox complexity <file>` analyzes a specific file
- [x] `aiox complexity --threshold 10` only shows functions above threshold
- [x] `aiox complexity --format json` outputs as JSON
- [x] `aiox complexity --top N` shows top N most complex functions
- [x] Calculates: if/else/for/while/switch/case/&&/|| counting
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
