# Story 13.1: Command Chaining & Pipelines

## Objective

Provide a CLI command to run multiple AIOX commands sequentially or in parallel, with timing and pass/fail reporting.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** to chain multiple commands together in a single invocation,
**so that** I can automate common workflows like lint+test+coverage.

## Acceptance Criteria

- [x] `aiox chain "lint,test,coverage --check"` runs commands sequentially, stops on first failure
- [x] `aiox chain --parallel "lint,typecheck"` runs commands in parallel
- [x] `aiox chain --continue-on-error "lint,test"` runs all even if one fails, reports at end
- [x] Reports timing and pass/fail for each step
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/chain.test.js`
