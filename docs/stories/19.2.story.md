# Story 19.2: Interactive Tutorial

## Objective

Provide a step-by-step tutorial for new AIOX users with progress tracking.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** new AIOX user,
**I want** an interactive tutorial that walks me through the basics,
**so that** I can learn the framework step by step.

## Acceptance Criteria

- [x] `aiox tutorial` starts step-by-step tutorial (prints all steps)
- [x] `aiox tutorial --step N` shows specific step
- [x] `aiox tutorial --list` lists all tutorial steps
- [x] `aiox tutorial --reset` resets tutorial progress
- [x] Tutorial covers: install, first story, agents, testing, deployment
- [x] Progress tracked in `.aiox/tutorial-progress.json`
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing

## Scope

**IN:** tutorial command, steps, progress tracking, reset
**OUT:** Interactive prompts, real-time execution
