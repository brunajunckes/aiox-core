# Story 27.3: Burndown Chart

## Objective

Provide a CLI command `aiox burndown` that renders an ASCII burndown chart showing story completion progress per sprint.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using AIOX,
**I want** to see a burndown chart for my sprint,
**so that** I can visualize remaining work and track progress against an ideal pace.

## Acceptance Criteria

- [x] `aiox burndown` shows burndown chart for current sprint (ASCII)
- [x] `aiox burndown --sprint N` shows burndown for specific sprint
- [x] `aiox burndown --format json` outputs as JSON
- [x] `aiox burndown --ideal` includes ideal burndown line
- [x] Reads story completion data from docs/stories/
- [x] Detects current sprint automatically from story filenames
- [x] ASCII chart with axis labels and legend
- [x] Zero external npm dependencies
- [x] Command registered in bin/aiox.js
- [x] 15+ tests in tests/cli/burndown.test.js

## Scope

**IN:** ASCII burndown chart, ideal line, sprint selection, JSON output
**OUT:** Interactive charts, time-series database, browser rendering
