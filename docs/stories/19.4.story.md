# Story 19.4: Changelog Viewer

## Objective

View recent changes from CHANGELOG.md or git log with filtering options.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a changelog viewer in the CLI,
**so that** I can quickly see what changed without leaving the terminal.

## Acceptance Criteria

- [x] `aiox changes` shows recent changes from CHANGELOG.md or git log
- [x] `aiox changes --since <tag>` shows changes since tag
- [x] `aiox changes --count N` limits to last N entries
- [x] `aiox changes --format json` outputs as JSON
- [x] `aiox changes --breaking` shows only breaking changes
- [x] Parses CHANGELOG.md or falls back to git log
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing

## Scope

**IN:** changes command, changelog parsing, git fallback, filtering
**OUT:** Changelog editing, auto-update
