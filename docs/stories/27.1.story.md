# Story 27.1: Milestone Tracker

## Objective

Provide a CLI command `aiox milestones` to track project milestones with progress percentages, target dates, and completion status.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** project manager using AIOX,
**I want** to track milestones with progress and target dates,
**so that** I can monitor project progress against planned deadlines.

## Acceptance Criteria

- [x] `aiox milestones add "<name>" --target "2026-05-01"` adds a milestone
- [x] `aiox milestones list` lists all milestones with progress
- [x] `aiox milestones update <id> --progress 75` updates progress percentage
- [x] `aiox milestones complete <id>` marks milestone as done (100%)
- [x] `aiox milestones remove <id>` deletes a milestone
- [x] `aiox milestones --format json` outputs as JSON
- [x] Store: `.aiox/milestones.json`
- [x] Progress bar rendered in ASCII for list view
- [x] Zero external npm dependencies
- [x] Command registered in bin/aiox.js
- [x] 15+ tests in tests/cli/milestones.test.js

## Scope

**IN:** Add, list, update, complete, remove milestones; JSON output; ASCII progress bar
**OUT:** Gantt charts, milestone dependencies, notifications
