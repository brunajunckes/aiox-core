# Story 27.4: Capacity Planner

## Objective

Provide a CLI command `aiox capacity` that calculates team velocity, shows sprint history, and forecasts completion for a given number of stories.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** project manager using AIOX,
**I want** to see team velocity trends and forecast story completion,
**so that** I can plan sprints and estimate delivery dates.

## Acceptance Criteria

- [x] `aiox capacity` shows team capacity: stories/sprint avg, velocity trend
- [x] `aiox capacity --forecast N` forecasts completion for N stories
- [x] `aiox capacity --format json` outputs as JSON
- [x] `aiox capacity --history` shows velocity per sprint
- [x] Calculates from docs/stories/ metadata
- [x] Detects velocity trend (increasing, decreasing, stable)
- [x] ASCII velocity history table with bars
- [x] Zero external npm dependencies
- [x] Command registered in bin/aiox.js
- [x] 15+ tests in tests/cli/capacity.test.js

## Scope

**IN:** Velocity calculation, forecast, history, JSON output
**OUT:** Resource allocation, team member tracking, calendar integration
