# Story 12.3: Git Hooks Manager

## Objective

Provide a CLI command to manage AIOX git hooks (pre-commit, pre-push) for automated quality checks.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** a git hooks manager that installs and manages quality-check hooks,
**so that** lint, typecheck, and tests run automatically before commits and pushes.

## Acceptance Criteria

- [x] `aiox githooks list` shows installed git hooks
- [x] `aiox githooks install` installs AIOX pre-commit and pre-push hooks
- [x] `aiox githooks remove` removes AIOX hooks
- [x] `aiox githooks status` shows which hooks are active and what they run
- [x] Pre-commit hook runs lint + typecheck
- [x] Pre-push hook runs tests
- [x] Hooks are shell scripts written to .git/hooks/
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/githooks.test.js`
