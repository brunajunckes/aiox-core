# Story 12.2: Dependency License Checker

## Objective

Provide a CLI command to list and validate dependency licenses from package.json and node_modules.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** a license checker that validates all dependency licenses,
**so that** I can ensure compliance with open-source licensing requirements.

## Acceptance Criteria

- [x] `aiox licenses` lists all dependency licenses from package.json + node_modules
- [x] `aiox licenses --check` exits 1 if any non-permissive license found (GPL, AGPL)
- [x] `aiox licenses --format json` outputs as JSON
- [x] `aiox licenses --allow MIT,ISC,...` sets allowed licenses
- [x] Reads license from node_modules/<pkg>/package.json license field
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/licenses.test.js`
