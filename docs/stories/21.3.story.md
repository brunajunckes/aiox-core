# Story 21.3: Code Search Engine

## Objective

Provide regex search across project files with type filtering, context lines, and multiple output formats.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a code search engine,
**so that** I can quickly find patterns across the codebase from the CLI.

## Acceptance Criteria

- [x] `aiox search <pattern>` regex search across project files
- [x] `aiox search <pattern> --type js` filters by file type
- [x] `aiox search <pattern> --ignore "node_modules,dist"` ignores directories
- [x] `aiox search <pattern> --context 3` shows N lines of context
- [x] `aiox search <pattern> --format json` outputs as JSON
- [x] `aiox search <pattern> --count` shows match count only
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
