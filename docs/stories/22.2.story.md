# Story 22.2: Code Duplication Finder

## Objective

Find duplicate code blocks (3+ consecutive identical lines) across the codebase.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a code duplication finder,
**so that** I can identify repeated code and refactor it into shared functions.

## Acceptance Criteria

- [x] `aiox duplicates` finds duplicate code blocks (3+ consecutive identical lines)
- [x] `aiox duplicates --min-lines 5` sets minimum block size
- [x] `aiox duplicates --format json` outputs as JSON
- [x] `aiox duplicates --type js` filters by file type
- [x] `aiox duplicates --threshold 80` sets similarity threshold %
- [x] Reports file pairs, line ranges, similarity score
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
