# Story 24.1: Resource Monitor

## Objective

Show CPU, memory, disk usage in a formatted table with watch mode and alert thresholds.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a resource monitor command,
**so that** I can check system resource usage directly from the CLI.

## Acceptance Criteria

- [x] `aiox resources` shows CPU, memory, disk usage in a formatted table
- [x] `aiox resources --watch` live refresh every 2 seconds
- [x] `aiox resources --format json` outputs as JSON
- [x] `aiox resources --alert` shows warnings when usage >80%
- [x] Uses os.cpus(), os.freemem(), os.totalmem(), child_process for disk
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
