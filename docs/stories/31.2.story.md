# Story 31.2 — Focus Mode

**Status:** InReview
**Sprint:** 31 — Developer Productivity
**Module:** `.aiox-core/cli/commands/focus/`

## Description

Implement `aiox focus` command to track focused work sessions with on/off toggling, status, history, and stats (today/week/month). State in `.aiox/focus-state.json`, history in `.aiox/focus-history.jsonl`.

## Acceptance Criteria

- [x] `aiox focus on` enables focus mode (records start time)
- [x] `aiox focus off` disables, shows duration
- [x] `aiox focus status` shows if active and for how long
- [x] `aiox focus history` shows focus sessions
- [x] `aiox focus stats` total focus time today/week/month
- [x] `aiox focus --format json` JSON output
- [x] Registered in `bin/aiox.js`
- [x] 15+ tests passing
