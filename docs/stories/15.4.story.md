# Story 15.4: Batch File Operations

## Objective

Implement batch file operations including rename, find, replace, and count with glob matching and dry-run support.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** batch file operations,
**so that** I can perform bulk renames, search/replace, and file discovery across the project.

## Acceptance Criteria

- [x] `aiox batch rename "*.test.js" --prefix "unit-"` batch rename files
- [x] `aiox batch find "TODO" --type js` find pattern in files by type
- [x] `aiox batch replace "oldText" "newText" --glob "src/**/*.js"` batch search/replace
- [x] `aiox batch count --type js` count files by type
- [x] `aiox batch --dry-run` preview without executing
- [x] 15+ tests in `tests/cli/batch.test.js`
- [x] Command registered in `bin/aiox.js`
