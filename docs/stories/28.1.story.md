# Story 28.1: API Endpoint Generator

## Objective

Generate REST endpoint boilerplate (handler + test) from a single CLI command, reducing repetitive setup for new API routes.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** to generate REST endpoint boilerplate with a single command,
**so that** I can quickly scaffold new API routes without manual copy-paste.

## Acceptance Criteria

- [x] `aiox gen-endpoint <name>` generates REST endpoint boilerplate (handler + test)
- [x] `aiox gen-endpoint <name> --methods GET,POST,PUT,DELETE` specifies HTTP methods
- [x] `aiox gen-endpoint --list` lists generated endpoints
- [x] `aiox gen-endpoint <name> --output <dir>` uses custom output dir
- [x] Generates handler.js + handler.test.js with proper exports
- [x] Shows help with no args or --help
- [x] Validates endpoint name is provided
- [x] Zero external npm dependencies
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing

## Scope

**IN:**
- Endpoint generation with configurable HTTP methods
- Handler + test file generation
- Listing generated endpoints
- Custom output directory support

**OUT:**
- Router integration
- Database connection boilerplate
- Authentication middleware injection
