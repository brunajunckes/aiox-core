# Story 17.2: File Cache Manager

## Objective

Provide a TTL-aware cache system for storing temporary values with automatic expiration, cache stats, and cleanup operations.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** CLI user and agent system,
**I want** a cache with TTL support, stats, and cleanup commands,
**so that** I can store temporary data that auto-expires and manage cache size.

## Acceptance Criteria

- [x] `aiox cache stats` shows cache size, file count, oldest/newest entry
- [x] `aiox cache clear` clears all cached files
- [x] `aiox cache clear --older-than 7d` clears files older than N days
- [x] `aiox cache list` lists cached files with sizes
- [x] `aiox cache set <key> <value>` caches a value with TTL
- [x] `aiox cache get <key>` gets cached value (null if expired)
- [x] Cache dir: `.aiox/cache/`
- [x] Zero external npm dependencies
- [x] 15+ tests passing

## Scope

**IN:**
- TTL-aware key-value caching
- Duration parsing (s/m/h/d)
- Stats and listing
- Age-based cleanup

**OUT:**
- File-based caching (binary blobs)
- LRU eviction
- Distributed cache
