## Story
**As a** developer new to the AIOX framework,
**I want** a CLI command that lists all available agents with descriptions and detail views,
**so that** I can discover which agents exist, understand their scope, and know how to activate them without reading documentation files manually.

## Acceptance Criteria

1. - [x] `aiox agents` command lists all available agents in a formatted table with columns: Name, Persona, Scope, Activation Syntax
2. - [x] `aiox agents --detail <name>` shows full detail for a specific agent: persona, commands (with `*` prefix), scope description, and activation example
3. - [x] Interactive mode: running `aiox agents` without flags offers an interactive picker where the user selects an agent from a numbered list
4. - [x] Agent data is sourced from `.aiox-core/development/agents/` directory — no hardcoded agent lists
5. - [x] Output is human-readable in terminal (no raw YAML/JSON dump)
6. - [x] Exit gracefully with helpful message if agent name in `--detail` flag does not match any known agent
