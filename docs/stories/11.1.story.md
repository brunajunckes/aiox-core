# Story 11.1: System Event Logger

## Objective

Provide a CLI command to log, view, filter, and watch system events stored in `.aiox/events.jsonl`.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** a system event logger that records and displays events,
**so that** I can observe what the system is doing and troubleshoot issues.

## Acceptance Criteria

- [x] `aiox events` shows last 20 system events from `.aiox/events.jsonl`
- [x] `aiox events --level error` filters by level (info, warn, error)
- [x] `aiox events --since 2h` shows events from last N hours
- [x] `aiox events --clear` clears the event log
- [x] `aiox events --tail` watches for new events in real-time
- [x] Event format: `{ timestamp, level, source, message, data }`
- [x] `logEvent(level, source, message, data)` exported utility for other commands
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/events.test.js`
