# Story 21.4: Code Snippet Manager

## Objective

Save, retrieve, search, and manage code snippets from the CLI.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a snippet manager,
**so that** I can save and reuse code snippets across projects.

## Acceptance Criteria

- [x] `aiox snippets list` lists saved snippets from `.aiox/snippets.json`
- [x] `aiox snippets add <name> --file <path> --lines 10-20` saves snippet from file
- [x] `aiox snippets add <name> --content "code here"` saves inline snippet
- [x] `aiox snippets get <name>` outputs snippet content
- [x] `aiox snippets remove <name>` deletes a snippet
- [x] `aiox snippets search <term>` searches snippet names and content
- [x] `aiox snippets export` exports all as JSON
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
