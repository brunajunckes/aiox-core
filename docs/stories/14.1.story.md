# Story 14.1: Project Analytics Dashboard

## Objective

Provide a CLI command to display project analytics: file counts, LOC by type, test/story/command/dependency counts, with JSON output and trend tracking.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** a project analytics dashboard,
**so that** I can see project health metrics at a glance.

## Acceptance Criteria

- [x] `aiox analytics` shows total files, LOC by extension, test/story/command/dependency counts
- [x] `aiox analytics --format json` outputs as JSON
- [x] `aiox analytics --trend` shows growth over time from `.aiox/analytics-history.jsonl`
- [x] Counts files by extension (js, ts, md, yaml)
- [x] Command registered in `bin/aiox.js`
- [x] 15+ tests in `tests/cli/analytics.test.js`
