# Story 23.4: Script Runner & Discovery

## Objective

List, search, run, and audit npm scripts from package.json.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a script runner and discovery tool,
**so that** I can find, run, and audit npm scripts efficiently.

## Acceptance Criteria

- [x] `aiox scripts` lists all npm scripts from package.json
- [x] `aiox scripts run <name>` runs npm script
- [x] `aiox scripts search <term>` searches scripts by name/command
- [x] `aiox scripts --format json` outputs as JSON
- [x] `aiox scripts audit` finds scripts with potential issues (missing deps, typos)
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
