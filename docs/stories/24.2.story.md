# Story 24.2: Build Time Tracker

## Objective

Record and display build times with trend analysis and averages.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a build time tracker,
**so that** I can monitor build performance and detect regressions.

## Acceptance Criteria

- [x] `aiox build-time` shows build times from `.aiox/build-times.jsonl`
- [x] `aiox build-time --record "<command>"` times a command and records it
- [x] `aiox build-time --trend` shows trend over last 10 builds
- [x] `aiox build-time --format json` outputs as JSON
- [x] `aiox build-time --clear` clears history
- [x] `aiox build-time --avg` shows average build time
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
