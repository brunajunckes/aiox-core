# Story 31.1 — Pomodoro Timer

**Status:** InReview
**Sprint:** 31 — Developer Productivity
**Module:** `.aiox-core/cli/commands/timer/`

## Description

Implement `aiox timer` command for Pomodoro-style time tracking with start, stop, status, and history subcommands. State persists in `.aiox/timer-state.json`, history in `.aiox/timers.jsonl`.

## Acceptance Criteria

- [x] `aiox timer start` starts 25min pomodoro timer
- [x] `aiox timer start --duration 15` custom duration
- [x] `aiox timer stop` stops current timer, records to history
- [x] `aiox timer status` shows remaining time
- [x] `aiox timer history` shows completed pomodoros
- [x] `aiox timer --format json` JSON output
- [x] State stored in `.aiox/timer-state.json`
- [x] History stored in `.aiox/timers.jsonl`
- [x] Registered in `bin/aiox.js`
- [x] 15+ tests passing
