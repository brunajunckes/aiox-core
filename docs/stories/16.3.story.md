# Story 16.3: API Documentation Generator

## Objective

Implement an API documentation generator via `aiox api-docs` that scans bin/aiox.js for registered commands and produces markdown or JSON documentation.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** to auto-generate API documentation from command metadata,
**so that** I can keep documentation in sync with the actual CLI commands.

## Acceptance Criteria

- [x] `aiox api-docs` generates API documentation from command metadata (markdown to stdout)
- [x] `aiox api-docs --format markdown` outputs as markdown
- [x] `aiox api-docs --format json` outputs as JSON
- [x] `aiox api-docs --output docs/api.md` writes to file
- [x] Scans bin/aiox.js for registered commands and extracts: name, description, story
- [x] Markdown output includes table with all commands
- [x] JSON output includes structured command data
- [x] 15+ tests in `tests/cli/api-docs.test.js`
- [x] Command registered in `bin/aiox.js`

- [x] Task 3: Implement JSON formatter
