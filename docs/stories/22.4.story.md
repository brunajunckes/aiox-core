# Story 22.4: Import Validator

## Objective

Validate all require() calls resolve to existing files, detect circular dependencies, and find unused imports.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** an import validator,
**so that** I can catch broken requires, circular dependencies, and unused imports.

## Acceptance Criteria

- [x] `aiox imports` validates all require() calls resolve to existing files
- [x] `aiox imports --circular` detects circular dependency chains
- [x] `aiox imports --unused` finds imported but unused modules
- [x] `aiox imports --format json` outputs as JSON
- [x] `aiox imports --fix` removes unused imports (dry-run by default)
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
