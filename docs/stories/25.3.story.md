# Story 25.3: Standup Report Generator

## Objective

Automatically generate daily standup reports by combining git commit history, in-progress stories, and blockers into a formatted report.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer participating in daily standups,
**I want** to auto-generate a standup report from my recent activity,
**so that** I can quickly share what I did, what I plan to do, and what blocks me.

## Acceptance Criteria

- [x] `aiox standup` generates a standup report with: yesterday's commits, today's stories, blockers
- [x] `aiox standup --format json` outputs report as JSON
- [x] `aiox standup --since 2d` configures lookback period (supports d/w/m)
- [x] `aiox standup --output report.md` writes report to file
- [x] Uses git log for "what I did" section
- [x] Uses story status (InProgress/Ready) for "what I'll do" section
- [x] Reads blockers from `.aiox/blockers.json`
- [x] Markdown formatting with sections
- [x] Zero external npm dependencies
- [x] 15+ unit tests covering all operations

## Scope

**IN:**
- Git log integration for recent commits
- Story directory scanning for in-progress work
- Blockers file reading
- Markdown and JSON output formats
- File output option
- Unit tests

**OUT:**
- Slack/Teams integration
- Automated posting
- Historical standup archive
