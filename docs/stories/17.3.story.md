# Story 17.3: Data Export/Import

## Objective

Enable full export and import of all AIOX data (config, profiles, aliases, history, kv-store) as a single JSON file for backup, migration, and diff operations.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** CLI user,
**I want** to export all my AIOX data as a single JSON and import it back,
**so that** I can migrate between machines or recover from data loss.

## Acceptance Criteria

- [x] `aiox data export` exports all AIOX data as single JSON to stdout
- [x] `aiox data export --output backup.json` writes to file
- [x] `aiox data import <file>` imports data from backup JSON
- [x] `aiox data import <file> --dry-run` previews import without writing
- [x] `aiox data diff <file>` shows differences between current state and backup
- [x] Zero external npm dependencies
- [x] 15+ tests passing

## Scope

**IN:**
- Multi-source data collection
- JSON export/import
- Dry-run preview
- Diff comparison

**OUT:**
- Binary file backup (use backup command)
- Incremental export
- Remote storage
