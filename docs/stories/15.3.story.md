# Story 15.3: Cron-like Scheduled Tasks

## Objective

Implement a cron-like scheduler that parses cron expressions, manages scheduled tasks, and executes matching tasks on a per-minute tick.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer using the AIOX CLI,
**I want** cron-like scheduled tasks,
**so that** I can automate recurring maintenance commands.

## Acceptance Criteria

- [x] `aiox cron list` lists scheduled tasks from `.aiox/cron.yaml`
- [x] `aiox cron add "<cron-expr>" "<command>"` adds scheduled task
- [x] `aiox cron remove <id>` removes by ID
- [x] `aiox cron run` starts scheduler (checks every minute)
- [x] `aiox cron next` shows next run times for all tasks
- [x] Simple cron parser for: `*/5 * * * *`, `0 * * * *`, etc.
- [x] 15+ tests in `tests/cli/cron.test.js`
- [x] Command registered in `bin/aiox.js`

## Tasks / Subtasks

- [x] Task 1: Implement cron field and expression parser
- [x] Task 2: Implement cronMatches and getNextRun
- [x] Task 3: Implement YAML-like file I/O for cron entries
- [x] Task 4: Implement list/add/remove/next subcommands
- [x] Task 5: Implement scheduler with tick and stop
- [x] Task 6: Register command in bin/aiox.js
- [x] Task 7: Write 15+ tests covering all functions
