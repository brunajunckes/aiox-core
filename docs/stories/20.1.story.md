# Story 20.1: Version Manager

## Objective

Manage project versioning: view current version, bump semver, preview changes, and inspect version history from git tags.

## Agent

@dev (Dex)

## Status

InReview

## Story

**As a** developer,
**I want** a CLI version manager,
**so that** I can bump and track project versions without manual edits.

## Acceptance Criteria

- [x] `aiox version` shows current version from package.json
- [x] `aiox version bump patch` bumps patch (5.0.3 -> 5.0.4)
- [x] `aiox version bump minor` bumps minor (5.0.3 -> 5.1.0)
- [x] `aiox version bump major` bumps major (5.0.3 -> 6.0.0)
- [x] `aiox version bump --dry-run` previews without writing
- [x] `aiox version history` shows version history from git tags
- [x] Updates package.json version field
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
