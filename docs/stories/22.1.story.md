# Story 22.1: Dead Code Detector

## Objective

Find exported functions that are never imported/required elsewhere in the codebase.

## Agent

## Status

## Story

**As a** developer,
**I want** a dead code detector,

- [x] `aiox dead-code --ignore "tests/**"` ignores patterns
- [x] Scans module.exports and require() calls to build usage map
- [x] Command registered in bin/aiox.js
- [x] 15+ tests passing
