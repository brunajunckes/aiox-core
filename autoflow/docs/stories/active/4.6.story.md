# Story 4.6 — Cost Tracking

**Epic:** AutoFlow Phase 1  
**Status:** DONE ✅  
**Date Completed:** April 10, 2026

---

## Description

Implement comprehensive cost tracking system: per-tenant billing, model pricing, quota enforcement, and monthly reports. Integrates with Paperclip for governance.

---

## Acceptance Criteria

- [x] Per-tenant cost tracking (separate ledger per customer)
- [x] Model pricing stored in PostgreSQL (Ollama $0, Haiku $0.80/1M, Opus $15/1M)
- [x] Real-time cost calculation on every LLM call
- [x] Quota enforcement via Paperclip budget gates
- [x] Monthly billing reports (CSV export)
- [x] Cost alerts when tenant approaches quota (80%, 100%)
- [x] Cost breakdown by workflow type (research, SEO, video)
- [x] Audit trail (all cost events logged)
