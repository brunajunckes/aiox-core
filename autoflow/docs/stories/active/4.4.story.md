## Technical Details

### Endpoints

```
POST /api/research
  { "query": "AutoFlow", "complexity": 8 }
  → { "results": [...], "sources": [...], "cost_usd": 0.12 }

POST /api/seo
  { "keyword": "AI automation", "locale": "pt-BR" }
  → { "volume": 1200, "difficulty": 45, "analysis": {...} }

POST /api/video
  { "script": "...", "style": "professional" }
  → { "scenes": [...], "duration": "15min", "spec": {...} }
```

### Authentication
- Bearer token in `Authorization` header
- Token validated against tenant table
- Tenant ID injected into request context

### Rate Limiting
- 100 requests/min per tenant
- Sliding window counter
- 429 response when exceeded
