## Technical Details

### Router Flow
```
Request → Complexity Score → Model Selection → Provider Call → Cost Log → Response
         ↓
      Ollama (local, fast)
         ↓ (if Ollama down)
      Claude (fallback, paid)
         ↓ (if both down)
      Error + Retry
```

### Complexity Scoring
- Score 1-5: Ollama (local)
- Score 6-10: Claude Haiku (cost-optimized)
- Score 11-15: Claude Opus (complex reasoning)

### Circuit Breaker States
- CLOSED: Normal operation
- OPEN: Provider down, reject immediately
- HALF_OPEN: Recovery test in progress
